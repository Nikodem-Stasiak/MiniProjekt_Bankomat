# 🏧 Symulator Bankomatu (Java)

Prosta, obiektowa aplikacja konsolowa symulująca działanie bankomatu, napisana w języku Java. Projekt stworzony w ramach nauki programowania obiektowego.

## 🚀 Funkcjonalności
* Otwieranie konta bankowego (z saldem początkowym lub pustego).
* Wpłacanie i wypłacanie gotówki.
* Sprawdzanie aktualnego salda.
* Zabezpieczenie przed wypłatą kwoty większej niż dostępne środki (obsługa wyjątków).
* Interaktywne menu w konsoli.

## 🛠️ Technologie
* **Język:** Java
* **Kluczowe zagadnienia:** Programowanie Obiektowe (OOP), Hermetyzacja, Przeciążanie konstruktorów, Wyjątki (`try-catch`, `throw`), Pętle (`while`), Instrukcja `switch`, Klasa `Scanner`.
* **Środowisko:** IntelliJ IDEA
